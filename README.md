This is an app to inform the user about the weather in Athens now, for the next 5 hours and the next 5 days

# You need apikey for the data from https://openweathermap.org/api 

# How to install it? 

choose a  folder in your pc 

open 
# git bash 

and write

# git clone https://github.com/Nikitas15K/weatherInAthens.git

inside the created project folder weatherInAthens, open a cmd, and press
# npm install

inside the project folder
# in the src/config/index.js
# insert the open weather api key inside the quotation mark:
# export const apiKey = ""

back to the cmd, press
# npm start

here we are!
